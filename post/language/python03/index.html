python的if,and,or

if

一个简单的实例：

cars = ['audi', 'bmw', 'subaru', 'toyota']
for car in cars:
    if car == 'bmw':
        print(car.upper())
    elif car == 'audi'
        print(car.lower())
    else:
        print(car.title())

判断条件

a == b #等于
a != b #不等于
a > b
a < b
a >= b
a <= b
(a > b) and (a = b) #与
(a > b) or (a = b)  #或
cat in cats #返回True/False
cat not in cats #返回True/False
if cats #判断是不是空的列表
