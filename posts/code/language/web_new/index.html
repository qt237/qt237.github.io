网站的基本结构

  • index.html文件:包含主页内容，第一次点进网站显示的内容。
  • images文件夹：网站上使用的所有图片。
  • styles文件夹：用于设置样式的CSS代码，比如说字体，背景颜色，文本等等。
  • scripts文件夹:包含所有网站的交互内容的JavaScript代码。
