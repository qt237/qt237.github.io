python的虚拟环境

虚拟环境是我们创建的一个位置，我们在里面安装包，达成一个包的隔离。有助于项目分离和依赖控制。

python -m venv ll_env

创建一个虚拟环境ll_env。一般来说都是.XXX的隐藏文件夹，这里没有隐藏。

source ll_env/bin/activate来激活。 激活后可以通过deactivate来关闭
